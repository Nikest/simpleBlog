# Simple Blog

#####Typescript based

Using technologies:
* React js
* Node js
* MongoDB

Environment:
* Webpack
* Docker

##Demonstration in Docker

```
docker compose build
docker compose up
```

##Starting on a local machine

* server
```
cd server
npm i
tsc src/index.ts && node src/index.js
```

_____

* frontend dev
```
cd frontend
npm i
npm run dev
```
or
```
npm run build
```
